HO08 - Hands-on Adobe Flex 3

Desde o início da Web tem sido uma de nossas obsessões trazer cada vez mais interatividade e dinamismo a experiência do usuário com a internet. Evoluímos desde páginas estáticas com puro HTML até aplicações ricas e dinâmicas com AJAX.

Porém, ainda assim, existem certas limitações e incompatibilidades que dificultam o desenvolvimento e manutenção destas aplicações, e somado a isto, falta ainda a realidade da riqueza de interações que as aplicações desktop proporcionam reproduzidos em um ambiente web.

Adobe Flex é a alternativa para essa necessidade e uma resposta para tais limitações. Sendo um framework RIA(Rich Internet Application), cross-browser, cross-plataform e open-source, Flex possibilita um rico dinamismo as interações do usuário com uma interface web. Tendo por trás uma simples linguagem de marcação MXML) unida de uma poderosa linguagem Orientada a Objetos(Action Script) é possível criar interfaces poderosas, simples, produtivas e encantadoras para fascinar seus usuários.
