Academia Web Developer


Html, CSS, JavaScript, JSP, Servlet, Struts, JavaServer Faces, Facelets, Ajax/RichFaces e JPA

A carreira Academia Web é voltada aos profissionais que já atuam com a plataforma Java, tais como programadores, arquitetos, analistas e gerentes técnicos, que desejam conhecer e trabalhar com os recursos web da plataforma Java Enterprise Edition (JEE), utilizando servlets e JavaServer Pages (JSP) e os principais frameworks do mercado.

Existe uma grande demanda no mercado por desenvolvedores que dominam as tecnologias JSP e servlets, e os principais frameworks para web e persistência. A Academia Web Developer, além de apresentar os conceitos fundamentais do desenvolvimento web, engloba também as tecnologias mais recentes como JSP 2.0, Servlets 2.4 e JSF 1.1/1.2 e os mais populares frameworks Java, com enfoque em Struts e Hibernate, além de mostrar a utilização de ferramentas essenciais como Ant, Tomcat, Eclipse, NetBeans e JasperReports.

Este treinamento é composto de seis cursos que abordam todas as partes do projeto de uma aplicação web. O aluno aprende a utilizar funcionalidades para web de ferramentas de desenvolvimento Java e a criar telas dinâmicas complexas usando várias tecnologias web. Além disso o aluno também aprende a realizar a persistência de dados com acesso a dados por meio do mapeamento objeto-relacional e se prepara para compreender e definir arquiteturas de aplicações web. Como um benefício adicional, o aluno estuda e pratica todos os tópicos da certificação oficial Sun Certified Web Component Developer (SCWCD).

O que eu vou aprender na Academia Web Developer ?

A Academia Web Developer é composta por 6 cursos especializados em Java, sendo que o primeiro curso AW1 - Desenvolvimento Avançado de Interfaces Web com HTML, CSS e JavaScript é focado em HTML, CSS e JavaScript, mas não da perspectiva de um Web Designer, mas sim de um programador Java.

O segundo curso AW2 Desenvolvimento Web com Java EE aborda as principais APIs Java EE para desenvolvimento Web, essenciais para a certificação Sun Certified Web Component Developer (SCWCD) : JavaServer Pages (JPS), Servlets e Bibliotecas de Tags abordando detalhadamente tópicos fundamentais como Expression Language, manipulação de sessão, etc.

O terceiro curso desta carreira AW3 - Desenvolvimento Web com Struts aborda um dos frameworks mais populares do mercado, o Jakarta Struts. Embora grande parte dos novos projetos não optem por utilizar Struts o número de projetos que já utilizam é muito grande.

O quarto curso desta carreira AW4 - Desenvolvimento com JavaServer Faces aborda detalhadamente este framework, que faz parte da especificação de Java EE 5, uma das principais escolhas dos arquitetos para desenvolvimento de aplicações Web atualmente.

A carreira tem continuidade com o curso de Hibernate AW5 - Desenvolvimento da camada de persistência com Hibernate complementa o desenvolvimento de aplicações Web abordando o mais importante framework de persistência Java.

E para finalizar apresentamos o curso de iReport e JasperReports AW6 - Desenvolvimento de relatórios.

Ou seja, você tem uma formação completa, respeita sua curva de aprendizado e está pronto para o mercado e para mais uma certificação Java.

Resultados e Objetivos


Ao concluir a carreira você estará apto:
  • Criar páginas web estáticas com HTML, CSS e JavaScript
  • Criar páginas dinâmicas com JSP e taglibs
  • Trabalhar com servlets e requisições e respostas HTTP
  • Empacotar aplicações web e instalá-las e gerenciá-las no Tomcat
  • Definir a arquitetura de aplicações, usando o padrão MVC (Model-View-Controller)
  • Utilizar o Struts para implementar aplicações web bem-estruturadas
  • Utilizar a tecnologia JSF para criar páginas dinâmicas baseadas em componentes e eventos
  • Programar a persistência de objetos usando o Hibernate
  • Criar relatórios e gráficos em ambiente web utilizando o JasperReports e o iReport
  • Utilizar os principais recursos de IDEs open source Eclipse e/ou NetBeans para o desenvolvimento web
