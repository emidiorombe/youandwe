Curso de Orientação a Objetos com ênfase em Java




"Aumentar reusabilidade, extensibilidade e manutenabilidade de softwares com relativa complexidade, dentro de um custo aceitável".

A Orientação a Objetos é a resposta para essas dificuldades! Mas como utiliza-la?

O objetivo do Curso de Orientação a Objetos é detalhar os principais elementos que compõem esse paradigma, aplicando código Java e UML para representação de modelos visuais, com o ponto de vista prático trazendo a realidade e os desafios do mercado.

O aluno aprende em detalhes:

  • Como funciona classes, atributos e métodos
  • O significado de uma instancia, e como o java gerencia um objeto em memória
  • Relacionar classes, diferenças entre Associação, Dependência, Agregação e Composição
  • O que é Herança e quando devemos usa-la
  • O poder do Polimorfismo
  • Interpretação de diagramas UML
  • Modelagem e implementação de classes de negócios

Todos esses conceitos aplicando código Java!

Este curso dá continuidade a preparação para a certificação SCJP por meio de muitas dicas, exemplos de questão e simulados.
