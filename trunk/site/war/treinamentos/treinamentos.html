A YaW em parceira com Globalcode oferece os melhores treinamentos na tecnologia Java.

Carreiras
A Academia Java é especialmente desenvolvida para quem deseja trabalhar profissionalmente com a tecnologia Java e obter uma certificação de programador.
Este treinamento é uma seleção dos tópicos mais importantes da Academia Java voltado para quem deseja dispor de menos tempo para o curso.
A Academia Web prepara o profissional para o desenvolvimento de aplicações Web usando Java e frameworks como Struts, JSF e Hibernate.
A Academia Enterprise prepara os desenvolvedores para a utilização do Java EE, apresentando conceitos usados em aplicações corporativas sofisticadas.
Em breve! Primeiro curso completo de formação de profissionais na área de desenvolvimento de softwares que abrange desde o básico até a carreira prática.

Core
Este treinamento apresenta um conjunto fundamental de conhecimentos para desenvolvedores de software que desejam iniciar a programação em Java.
Este treinamento é especializado no desenvolvimento de aplicações Web usando tecnologia Java, especialmente JavaServer Faces (JSF).
Treinamento focado no desenvolvimento de componentes EJB da plataforma Java EE. São tratados asssuntos como EJB 2.x e 3.x, JPA e Web Services.
Aborda o assunto design-patterns de forma teórica, prática e objetiva. São apresentados 33 design-patterns das famílias GoF e Java EE.
Para quem pretende entender mais sobre a importância dos requisitos e das técnicas mais importantes para documentação e coleta dos requisitos.

Hands-on
Você aprenderá a criar interfaces poderosas, simples, produtivas e encantadoras para fascinar seus usuários.
O aluno utiliza as bibliotecas RichFaces e Ajax4JSF para criar uma aplicação JSF e AJAX fazendo desde a configuração do projeto até a execuçÃo da aplicaçÃo.
O aluno aprende a criar a camada de persistência de uma aplicaçÃo usando JPA, o padrão de persistência do Java EE.
O aluno cria uma aplicação JBoss Seam, usando POJO, anotações de persistência e validação, e recursos de produtividade.
