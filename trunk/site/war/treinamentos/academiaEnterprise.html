Academia Enterprise


Formando profissionais para desenvolvimento de soluções críticas

A Academia Enterprise prepara os desenvolvedores para a utilização do Java Enterprise Edition (Java EE), apresentando conceitos usados em aplicações corporativas sofisticadas, com grandes exigências de disponibilidade, robustez e acesso concorrente.

Esta carreira aborda tecnologias já consolidadas como EJB 2.1 e outras tecnologias mais recentes que fazem parte do Java EE 5, como EJB3, JPA e JAX-WS. No cenário tecnológico atual, uma das palavras-chave é integração de sistemas, e esta carreira enfoca também as tecnologias mais importantes dessa área, como JMS, XML e Web Services.

Além disso, o curso também é preparatório para a certificação SCBCD 5.0, que aborda EJB 3 e JPA, sendo apresentadas diversas questões no estilo da prova.

Resultados e Objetivos


Ao concluir a carreira você estará apto:
  • Entender e implementar componentes de negócio utilizando EJB 2.x e EJB 3
  • Integrar componentes EJB a aplicações web
  • Criar Web Services com e sem EJB
  • Usar JMS para integração via servidores de mensagens
  • Fazer o empacotamento e a instalação de aplicações enterprise no JBoss
  • Aplicar os conceitos básicos das APIs RMI e TCP/IP ao desenvolvimento corporativo com Java
  • Trabalhar com servidores LDAP via JNDI
