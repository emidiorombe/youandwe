A Carreira Intensivo Academia Java

Agora com Java 1.4 e Java 5, NetBeans e Eclipse

A carreira Intensivo Academia do Java é uma seleção de tópicos da Academia do Java que foi especialmente desenvolvida para atender profissionais que desejam conhecer e trabalhar com o Java Standard Edition (Java SE) e desenvolver aplicações web com Java mas não têm disponibilidade de tempo para cursar a Academia do Java completa.

Essa carreira é composta por quatro cursos de Java que abrangem desde a sintaxe da linguagem e técnicas de orientação a objetos, até a construção de uma aplicação web corporativa.

O mercado de trabalho para profissionais Java mantém-se excelente e o número de projetos baseados nessa tecnologia continua em franco crescimento. Na carreira Intensivo Academia do Java, o aluno aprende na prática a desenvolver aplicações completas com a tecnologia Java, realizando atividades fundamentais em projetos reais, como:

  • Interpretação de diagramas UML
  • Modelagem e implementação de classes de negócios
  • Integração com bancos de dados relacionais utilizando Java e JDBC
  • Desenvolvimento web em três camadas com Servlets e JavaServer Pages (JSPs)

Preparação para certificações Java

Um dos grandes desafios encontrados depois de estudar uma nova tecnologia é adquirir experiência. Como conseguir trabalhar profissionalmente em projetos Java sem experiência prévia? Ter uma certificação oficial pode aumentar as suas chances!

O aluno do Intensivo Academia do Java recebe todo o material necessário para se preparar para as certificações oficiais Java. Grande parte dos tópicos exigidos pela certificação Sun Certified Java Programmer (SCJP) 1.4 é abordada em sala de aula. Os tópicos presentes na certificação não estudados nessa carreira são Inner classes, I/O, Threads e Assertions (estes são cobertos apenas na carreira Academia do Java completa).

Alunos do Intensivo Academia do Java ganham um benefício adicional: podem assistir ao curso de preparação para a certificação SCJP (curso AJ7 da Academia do Java) quantas vezes quiserem no decorrer de um ano.

Qual a diferença dos cursos intensivos?

O modo intensivo diferencia-se pela periodicidade diária contínua, portanto, com período de duração abreviado. Este formato é recomendado para quem tem urgência em capacitar-se em algum segmento específico da tecnologia Java. Disponibilizamos neste formato os principais cursos que fazem parte das Academias. Os cursos intensivos são ministrados de segunda a sexta-feira ou de segunda a quinta-feira, período matutino ou noturno.

Resultados e Objetivos


Ao concluir a carreira você estará apto:
  • Utilizar os principais recursos oferecidos pela linguagem Java
  • Aplicar os conceitos de orientação a objetos usando a linguagem Java e suas APIs
  • Utilizar coleções para armazenamento e organização de dados em memória
  • Criar e interpretar modelos orientados a objetos em diagramas de classes UML, e criar diagramas de casos de uso e de seqüência
  • Implementar o tratamento de erros na sua aplicação
  • Criar componentes de acesso a banco de dados utilizando JDBC e o design pattern DAO (Data Access Object)
  • Implementar aplicações web utilizando Servlets e JavaServer Pages (JSP)
  • Utilizar o servidor Apache Tomcat para instalar e testar aplicações Java para web
  • Desenvolver e gerenciar produtivamente seus projetos com o NetBeans ou o Eclipse
